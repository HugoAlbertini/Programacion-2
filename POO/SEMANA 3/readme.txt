💻 CASOS PRÁCTICOS DESARROLLADOS
================================================================================

1. 👨‍🎓 REGISTRO DE ESTUDIANTES
   ─────────────────────────────
   • Clase: Estudiante
   • Atributos: nombre, apellido, curso, calificación
   • Métodos: mostrarInfo(), subirCalificacion(), bajarCalificacion()
   • Funcionalidad: Gestión completa de información estudiantil

2. 🐕 REGISTRO DE MASCOTAS
   ──────────────────────
   • Clase: Mascota
   • Atributos: nombre, especie, edad
   • Métodos: mostrarInfo(), cumplirAnios()
   • Funcionalidad: Simulación del crecimiento de mascotas

3. 📖 ENCAPSULAMIENTO CON LIBROS
   ────────────────────────────
   • Clase: Libro (con atributos privados)
   • Atributos: titulo, autor, añoPublicacion
   • Métodos: Getters completos + Setter con validación
   • Funcionalidad: Protección de datos mediante encapsulamiento

4. 🐔 GESTIÓN DE GALLINAS EN GRANJA DIGITAL
   ────────────────────────────────────────
   • Clase: Gallina
   • Atributos: idGallina, edad, huevosPuestos
   • Métodos: ponerHuevo(), envejecer(), mostrarEstado()
   • Funcionalidad: Simulación de granja con múltiples gallinas

5. 🚀 SIMULACIÓN DE NAVE ESPACIAL
   ─────────────────────────────
   • Clase: NaveEspacial
   • Atributos: nombre, combustible
   • Métodos: despegar(), avanzar(), recargarCombustible(), mostrarEstado()
   • Funcionalidad: Sistema completo de navegación espacial con validaciones
